## Run locally
Clone the project

```terminal
  git clone https://github.com/Ayngie/assignment-javascript-ToDo.git
```

Go to the project directory

```terminal
  cd assignment-javascript-ToDo
```

Install dependencies

```terminal
  npm install
```

Open in Live Server
